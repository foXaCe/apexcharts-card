# ApexCharts Card

A highly customizable graph card for Home Assistant's Lovelace UI, based on
[ApexCharts.js](https://apexcharts.com).

## Features

- Line, area, bar, scatter, pie, donut and radial bar chart types
- Multiple Y axes, annotations and color thresholds
- Statistics, `group_by` and aggregation functions
- Brush charts, configuration templates and experimental features

See the [README](https://github.com/foXaCe/apexcharts-card) for the full list of
configuration options.
